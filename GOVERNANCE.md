# Governance

Appwright is currently maintained by **Bhargav Andhe** (`@bhargavandhe`).

The maintainer is responsible for API direction, releases, security response, compatibility
policy, and repository administration. Significant architectural decisions should be recorded as
ADRs and discussed publicly before implementation when practical.

Additional maintainers may be added after sustained, high-quality participation and demonstrated
alignment with the project's technical and community standards. Maintainer changes are recorded
in this file and `CODEOWNERS`.

During the pre-1.0 period, the project prioritizes correctness and a coherent API over backward
compatibility. Breaking changes must still be documented in pull requests and the changelog.
