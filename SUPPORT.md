# Support

Appwright is alpha software maintained on a best-effort basis.

## Usage questions

Use [GitHub Discussions](https://github.com/bhargavandhe/appwright/discussions) for setup help,
API questions, design proposals, and general conversation.

## Bugs and feature requests

Use [GitHub Issues](https://github.com/bhargavandhe/appwright/issues) after searching existing
reports. Include a minimal reproduction and the output of `appwright doctor --json` when relevant.

## Security

Follow [SECURITY.md](SECURITY.md). Never place vulnerability details, access keys, private traces,
or sensitive application screenshots in a public issue.

There is currently no paid support or guaranteed response-time service.
