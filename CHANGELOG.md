# Changelog

All notable changes to Appwright will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and releases use
[Semantic Versioning](https://semver.org/spec/v2.0.0.html). Before 1.0, minor releases may contain
documented breaking changes.

## [Unreleased]

### Added

- Initial Android-first Appium 3 framework foundation.
- Playwright-style lazy locators, strict resolution, actionability, and retrying assertions.
- Generated synchronous and asynchronous Python facades.
- Typed Pydantic configuration, selector, trace, capability, and diagnostic models.
- Managed local and remote Appium connections.
- Pytest fixtures, failure artifacts, CLI diagnostics, and bounded trace archives.
